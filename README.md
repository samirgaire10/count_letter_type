# count_letter_type
count letter type



## Run Locally

Clone the project

```bash
  git clone https://github.com/samirgaire10/count_letter_type.git
```

Go to the project directory

```
  cd count_letter_type 
```


Compile the C file  
```
gcc -o count_letter_type.o count_letter_type.c
```

Run
```
./count_letter_type.o
```
